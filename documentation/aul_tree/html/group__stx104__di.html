)

Apex Univeral Library STX104 specific function that retrieves the specified digital input (DI) channel.

Parameters
[in]boardThe descriptor obtained from the function AUL_Device_Open.
[in]channelThe channel number that the data value will be read from; STX104_DI0 (0) <= channel <= STX104_DI3 (3) (i.e. STX104_DI_ENUM);
[out]valuePointer to the variable that will receive the digital input value
Returns
An error code of type int is returned. If successful, returns 0 (SUCCESS), otherwise it returns a non-zero error code. Please refer to AUL_STATUS_ENUM for all the possible error codes.
