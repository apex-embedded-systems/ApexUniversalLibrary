Installation

Driver

Refer to driver installation README.txt within the "drv_aul_x86" folder for further details.

The driver and library can be unpacked and placed almost anywhere. Initially, we suggest installation in your home folder. We usually name the folder to unpack to be "aes_eclipse", but it can be anything you wish.

A quick session that is typical.

  1. Remove previously compiled files:

    user:~/aes_eclipse/drv_aul_x86$ make clean

  1. Build the new driver:

    user:~/aes_eclipse/drv_aul_x86$ make

  1. Install the driver:

    user:~/aes_eclipse/drv_aul_x86$ sudo insmod mod_auldrv.ko offset=0x300,0x310 bytes=0x10,0x10 acronym="stx104","tracere" type="stx104","tracere"

  1. See if the driver module is listed in the kernel:

    user:~/aes_eclipse/drv_aul_x86$ lsmod 

    Module                  Size  Used by
mod_auldrv             34432  0

Note: your specific installation might require different base addresses.
