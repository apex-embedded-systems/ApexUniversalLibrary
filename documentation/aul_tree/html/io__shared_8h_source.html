102  /* acronym
103  Same name given to file nodes. For example, if we have two tracere boards,
104  then the acronym will be "tracere0" and "tracere1" where the minor number
105  is appended to the generic acronym. If only a generic acronym is available
106  and sent to the driver to retrieve information, it will pull the information
107  for the device that is not active. For example, if we send "tracere" and
108  we have "tracer0" and "tracer1", and "tracer0" has already been open, it will
109  then simply return the information for "tracer1" and update the acronym to
110  support it. If we had sent "tracer0" and it was open and we try to open with
111  it, it will return an error (i.e. IO_Definition_Get() ).
112 
113  <acronym><minor_number>
114  */
115  char acronym[IO_DEFINITION_ACRONYM_SIZE];
