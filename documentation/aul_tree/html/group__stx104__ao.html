Apex Universal Library STX104 specific function that sets both analog output channels simultaneously (to minimize skewing between the two channels).

Parameters
[in]boardThe descriptor obtained from the function AUL_Device_Open.
[in]value_ao_aUnsigned 16-bit integer value to write out to the analog output channel-A [J7.17 (DAC_OUT_1)]
[in]value_ao_bUnsigned 16-bit integer value to write out to the analog output channel-B [J7.16 (DAC_OUT_2)]
Returns
An error code of type int is returned. If successful, returns 0 (SUCCESS), otherwise it returns a non-zero error code. Please refer to AUL_STATUS_ENUM for all the possible error codes.

Apex Universal Library STX104 specific function that sets a single analog output channel.

Parameters
[in]boardThe descriptor obtained from the function AUL_Device_Open.
[in]channelThe channel number that the data value is to be written out to; STX104_AO_A (0) <= channel <= STX104_AO_B (1) (i.e. STX104_AO_ENUM).
[in]valueUnsigned 16-bit integer value to write out to the analog output channel.
Returns
An error code of type int is returned. If successful, returns 0 (SUCCESS), otherwise it returns a non-zero error code. Please refer to AUL_STATUS_ENUM for all the possible error codes.
