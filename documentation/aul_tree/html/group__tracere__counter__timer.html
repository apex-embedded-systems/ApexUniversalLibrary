Parameters

[in]rwIndicates if the control word is a write or read instruction. See I8254_RW_ENUM.
[in]modeIndicates the mode of operation. See I8254_MODE_ENUM.
[in]bcdIndicates whether the data is of BCD or binary type. See I8254_TYPE_ENUM.
[out]i8254_control_wordPointer to the location in which the control word is to be written.
Returns
