AUL_DLL_API int AULCALLCONV Tracere_Restart (int board, TRACERE_RESTART_ENUM value)
 This function is used to commit initialization data (i.e. configurations) to the hardware and also restart hardware into a known state. For example TRACERE_RESTART_COLD_DIO will commit any digital I/O related configuration and put the hardware into a known state as defined by the configuration parameters. In all cases, a COLD or WARM reset will do the same things in part because the underlying hardware configurations do not have read-back capability and so a warm start is not possible. A WARM restart is the case where existing configuration data is read back from the hardware and the software begins at that point. It is possible to restart the entire device (board) or just sub-sections (components). More...
