HTML File Template Example

This example demonstrates how the HTML file template feature works. This feature makes it possible to put Doc-O-Matic generated content into your site layout effortlessly.

Preparation

Prepare a HTML file that includes all resources the Doc-O-Matic generated content will require. This includes the Doc-O-Matic style sheet, the JavaScript file. Also setup body onload and onmousedown code so the content gets initialized correctly.

At any location in the HTML file you place the HTML content placeholders %%DOC-O-MATIC_CONTENT%%. Doc-O-Matic will insert its content at this location. That means that the content of the template wraps around Doc-O-Matic's content.

Additionally you can add the %%DOC-O-MATIC_SCRIPTS%% placeholder in the HEAD section of the HTML template. This placeholder will be replaced by HTML code that includes necessary CSS and JavaScript files that Doc-O-Matic generates.

Output Design

There are no limitations as to what kind of content your template can contain. This feature is perfect for seamless integration of Doc-O-Matic output into the design you're already using.

For configuration details please see the documentation.
