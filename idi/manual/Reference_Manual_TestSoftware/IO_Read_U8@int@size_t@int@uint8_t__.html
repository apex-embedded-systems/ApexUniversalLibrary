int IO_Read_U8(int board_id, size_t line, int location, uint8_t * value);
File

idi.c

Returns

A zero (SUCCESS) is returned if successful, otherwise a negative error code is returned.

Description

brief Reads uint8_t from I/O port. Macros are used to guide the target implementation. 

param[in] location the enumerated register symbol. The enumerated symbol is composed of offset and bank information used to determine the final address information. param[in] value The pointer to the destination of the read uint8_t value.
